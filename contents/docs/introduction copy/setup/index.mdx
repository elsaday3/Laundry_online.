Pembuatan aplikasi ini mengunakan teknologi : 
## Teknologi yang Digunakan
1. Frontend: HTML untuk struktur halaman, CSS untuk desain, dan Bootstrap untuk tampilan responsif.
2. Backend: PHP untuk proses server dan aplikasi.
3. Basis Data: MySQL untuk menyimpan data pengguna transaksi dan pembayaran.
4. Development: proses pembuatan dan pengujian aplikasi, menggunakan XAMPP untuk menjalankan server lokal
