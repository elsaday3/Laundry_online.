## Software Requirements untuk website Laundry Express 

## 1.  Web Server
  * Apache HTTP Server atau Nginx: Untuk mengelola permintaan HTTP dan mendistribusikan halaman website ke pengguna

## 2. Database Management System
  * MySQL atau PostgreSQL: Digunakan untuk menyimpan data pelanggan, pesanan, transaksi, dan status laundry

## 3. Programming Languages
  * PHP: Digunakan untuk pengembangan backend dan pemrosesan server.
  * HTML5: Untuk struktur dasar halaman website.
  * SQL: Untuk query dan pengelolaan database.
