## System Requirements

## 1. Fitur Sistem 
  * Sistem Pemesanan: Pelanggan dapat memesan layanan laundry melalui aplikasi dengan memilih jenis layanan dan waktu pengambilan/pengantaran.
  * Pembayaran Online: Integrasi dengan gateway pembayaran untuk transaksi online, seperti OVO, GoPay, atau transfer bank.
  * Sistem Laporan Keuangan: Fitur untuk mencatat pemasukan dan pengeluaran yang terhubung dengan sistem akuntansi.

## 2. Backup Data Rutin 
  * Melakukan backup data pelanggan dan transaksi secara otomatis

## 3. Keamanan
  * Enkripsi Data Sensitif: Untuk melindungi data pribadi dan transaksi pelanggan

## 4. Platform Sistem
  * Web-based Application: Dapat diakses melalui browser di perangkat desktop, laptop, atau mobile.
